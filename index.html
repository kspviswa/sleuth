sleuth

Code names as S-PIN (Scripted PIN ) is an instrumentation / debugger tool, created on top of Intel's Pin. Slueth exposes a scripting framework on top of PIN's API, so that developers / sluethers can exploit the rich PIN APIs in a programmable perspective. It is aimed to provide rich scripting features like dtrace of solaris / unix.

Pin

Pin is a dynamic binary instrumentation framework for the IA-32 and x86-64 instruction-set architectures that enables the creation of dynamic program analysis tools.

For more info, visit. https://software.intel.com/en-us/articles/pin-a-dynamic-binary-instrumentation-tool
